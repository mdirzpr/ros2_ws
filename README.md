# ros2_ws
ROS2 Workspace Template

## Setup

First, clone this repository.

```
git clone https://github.com/mdirzpr/ros2_ws.git
```
Then, build the Docker image.

```
docker compose build
```

Start a container.

```
docker compose run ros2_jazzy
```
